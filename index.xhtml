WebPhyloTree

A brief introduction to programming (in Chinese)

Read here: https://github.com/KelvinLu1024/Programming-Tutorial-Chinese/blob/master/tutorial.ipynb

I wrote it for my girlfriend. She once attended an introductory course on programming, but frustrated by controlling graphic user interface imperatively and the concept of object-oriented programming. So I wrote this article to help her understand the most essential concepts in procedural programming (expression, value, statement, definition, if-statement, for-loop, while-loop, etc.). Also, I want to proof with this introductory text that programming can be fun and easy to learn.
